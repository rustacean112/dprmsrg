# dprmsrg
Türkiye'deki depremler için için Kandilli Rasathanesi'nden bilgi çeken bir kabuk komut.

# Nasıl İndirilir ve Kurulur?
İlk önce:

    git clone https://github.com/Huso112/dprmsrg.git

komutunu giriyoruz.

İkinci olarak:

    cd ~/dprmsrg && sh kur.sh

komutunu yazıyoruz.

İlk önce:

    dprmsrg
komutunu yazalım.
